# AgriCalc Pakistan

AgriCalc Pakistan is a static, SEO-friendly agriculture website built with HTML, CSS, and vanilla JavaScript. It provides crop guides, farming tools, calculator pages, and policy content suitable for GitHub Pages deployment.

## Features
- Modern responsive design
- Dark mode support
- Agriculture calculators
- Crop guide pages
- Blog listing page
- Policy pages for privacy, terms, disclaimer, and cookies
- 404 error page
- Static site ready for GitHub Pages and Cloudflare

## Folder Structure
- `index.html`
- `about.html`
- `contact.html`
- `privacy-policy.html`
- `terms.html`
- `disclaimer.html`
- `cookie-policy.html`
- `404.html`
- `robots.txt`
- `sitemap.xml`
- `manifest.json`
- `README.md`
- `assets/css`
- `assets/js`
- `assets/images`
- `tools`

## Deployment
1. Push the repository to GitHub.
2. Enable GitHub Pages in repository settings.
3. Choose the `main` branch or the branch containing the site.
4. Set the site to publish from the repository root.

## Notes
- No backend dependencies required.
- All pages are static and use client-side JavaScript for interaction.
